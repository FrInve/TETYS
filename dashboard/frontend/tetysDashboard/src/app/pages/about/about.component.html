About

The Next Generation Internet Search project represents a pivotal European initiative aimed at transforming how people search and discover information online. This project supports a broad spectrum of individuals to challenge and redefine the conventional methodologies of internet search.

Tetys stands for Topics Evolution That You See and it is a tool to explore specific topics and see how these have been addressed by scientific research over time. The data was downloaded from Scopus API between June 1 and 30, 2024 via api.elsevier.com and SCOPUS.
